2D controls

mouseScroll: zoom in/out

mouseClick +mouseMove: XY translation

[CmdKey or CtrlKey] + MouseClick +mouseMove: Z rotation


3D controls

mouseScroll: zoom in/out

mouseClick + mouseMove: XYZ rotation

[CmdKey or CtrlKey] + MouseClick +mouseMove: XY translation

[ShiftKey] + MouseDrag: Select many nodes together
